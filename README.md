# ruby-pong
Ruby 2P version of the classic arcade game using GOSU.
No score cap. Play endlessly for now.

#To play:

gem install gosu

ruby pong.rb

------------------------------

#Controls:
Designed to be used with my arcade controller

Left side: use 'upArrow' and 'downArrow'
Right side: use 'f' and 'v'

'Esc' to close.
